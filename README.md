## Getting Started

1. **Open the page**: Simply open `index.html` in your web browser
2. **Customize content**: Edit the text, colors, and product information in `index.html`
3. **Modify styles**: Update `styles.css` to match your brand colors and design
4. **Extend functionality**: Add more JavaScript features in `script.js`

## Responsive Breakpoints

- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: 600px - 767px
- **Small Mobile**: Below 600px

## Customization Guide

### Colors
Edit the CSS variables at the top of `styles.css`:
```css
:root {
    --primary-color: #0066ff;
    --secondary-color: #00d4ff;
    --dark-color: #1a1a2e;
    --light-color: #f0f2f5;
}
```

### Products
Update product information in the Products section of `index.html`

### Contact Information
Replace placeholder contact details with your actual information in the Contact section

### Branding
- Update the logo text and icon in the navbar
- Change the company name "PowerShield" throughout
- Replace colors with your brand palette
